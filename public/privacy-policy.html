Kebijakan Privasi

Terakhir diperbarui: 4 Februari 2026

Selamat datang di Montana ID Sync. Kami berkomitmen untuk melindungi data pribadi Anda melalui protokol keamanan tinggi. Kebijakan ini menjelaskan data apa yang kami ambil melalui aplikasi kami.

1. Data yang Kami Kumpulkan

Sesuai dengan alur verifikasi sistem kami, kami mengumpulkan data berikut:

  • Identitas Dasar: Nama lengkap dan alamat email (melalui Google Sign-In atau input manual).
  • Komunikasi: Nomor WhatsApp untuk keperluan sinkronisasi laporan stok dan alert sistem.
  • Geolokasi (GPS): Kami mengakses lokasi presisi Anda melalui Geofencing Protocol untuk memastikan verifikasi di lokasi yang diizinkan.
  • Data Biometrik: Penggunaan kamera untuk Digital Face Signature. Data ini digunakan secara real-time untuk verifikasi identitas.

2. Penggunaan Google User Data

Aplikasi kami menggunakan layanan Google OAuth. Kami hanya mengakses profil publik (nama dan email) untuk mempercepat proses registrasi tanpa menyimpan password Google Anda di server kami.

3. Keamanan Data (Secure Protocol)

Kami menggunakan enkripsi SHA-256 (Secure Hash) untuk melindungi informasi sensitif. Data lokasi dan biometrik diproses untuk validasi sesi dan tidak disalahgunakan untuk pelacakan di luar kepentingan operasional sistem Montana ID Sync.

4. Penyimpanan Data

Data Anda disimpan secara aman selama Anda aktif menjadi bagian dari ekosistem Montana ID. Anda berhak meminta penghapusan data dengan menghubungi administrator sistem.

5. Persetujuan (Pakta Integritas)

Dengan melanjutkan penggunaan aplikasi ini dan mencentang persetujuan pada langkah "Pakta Integritas", Anda secara sadar memberikan izin kepada kami untuk memproses data tersebut di atas.
